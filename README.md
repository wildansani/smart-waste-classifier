#Smart Waste Classifier

Klasifikasi sampah organik vs anorganik menggunakan Deep Learning dengan arsitektur **EfficientNetB3** dan Transfer Learning.

Dataset
- **Sumber**: Kaggle - Waste Classification Data
- **Total gambar**: 25.077 gambar
- **Kelas**: Organik dan Anorganik
- **Distribusi**: Seimbang (50:50)
- Link Data Set
- https://www.kaggle.com/datasets/techsash/waste-classification-data?resource=download



Tech Stack
- TensorFlow / Keras
- EfficientNetB3 (Transfer Learning)
- Google Colab (GPU T4)
- Streamlit (Deployment)
